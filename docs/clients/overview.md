# Clients

Phlix is available on a wide range of platforms — from living-room TVs to phones to the web browser on your laptop.

## Available Clients

| Platform | App | Notes |
|----------|-----|-------|
| [Windows](./windows) | Native app | Full media experience |
| [macOS](./macos) | Native app | Full media experience |
| [Linux](./linux) | Native app | Full media experience |
| [Mobile](./mobile) | iOS & Android | Stream on the go |
| [Web](./web) | Browser-based | No install needed |
| [Roku](./roku) | Channel app | For Roku devices |
| [Samsung Tizen](./tizen) | Smart TV app | For Samsung Smart TVs |

## Skip Button

Phlix supports the [Skip Button protocol](./skip-button-integration-brief) for hands-free navigate during intros and recaps.
